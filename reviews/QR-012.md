## Generated C Quality

Three new functions in `emit.py`:

1. **`_emit_switch_stmt`** — emits `switch (cond) { ... }`, buffers condition
   expression and flushes pre_stmts (same pattern as the OZ-005 return fix).
   Handles CompoundStmt body correctly.

2. **`_emit_case_stmt`** — emits `case <expr>:` followed by body statements.
   Uses `_emit_expr` for the case value, supporting both integer literals
   and enum constants via `ConstantExpr`.

3. **`DefaultStmt` handling** — inline in `_emit_stmt`, emits `default:`
   followed by inner statements.

Also added **`ConstantExpr` support** in `_emit_expr` — unwraps to the inner
expression (typically a `DeclRefExpr` for enum constants). This is the key
piece that makes enum constants work in case labels.

The implementation is clean and follows the existing patterns in emit.py.
