## Generated C Quality

The fix is a single-line change: `header_name.rsplit(".", 1)[0]` to `Path(header_name).stem`.

Before (buggy):
```python
stem = header_name.rsplit(".", 1)[0]
# "services/PXAppConfig.h" → "services/PXAppConfig" → #include "services/PXAppConfig_ozh.h"
```

After (fixed):
```python
stem = Path(header_name).stem
# "services/PXAppConfig.h" → "PXAppConfig" → #include "PXAppConfig_ozh.h"
```

Edge cases verified manually:
- Deep nesting (`a/b/c/Foo.h` → `Foo`) — correct
- Flat path (`Foo.h` → `Foo`) — correct, no regression
- Dotted filename (`Foo.Bar.h` → `Foo.Bar`) — correct

The fix is minimal and correct. Exactly what a human would write.
